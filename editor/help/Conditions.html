Conditions

Logical conditions, describing when an object/event can be activated.

Format:

<cond check="is">
  <type>...</type>
  <param>...</param>
</cond>
Attributes:
  • check - when the related object will be enabled? Available values:
    • is (default) - the condition is fulfilled.
    • isnt - the condition is not fulfilled.
    • once - the condition was fulfilled.
    • never - the condition wasn't fulfilled.
Parameters:
  • type - condition type. Available values:
    • none - empty condition. Parameter: none.
    • clear - all physicals of given type were wiped out. Parameter: filter name (string).
    • time-elapsed - given amount of time elapsed after the map had been loaded. Parameter: time (float, [s]).
    • region-triggered - somebody walked into a given region. Parameter: region name (string).
    • killed - named physical is dead and his corpse is still on the map. Parameter: physical name (string).
  • param - depends on type parameter.
